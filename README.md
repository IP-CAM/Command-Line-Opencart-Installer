Command Line Opencart Installer
=====================

A command line installer for opencart.


Summary
-------

Can be used under linux, or windows with cygwin

Can also pre-install templates if specified


Known Issues
------------

* Only Journal template supported so far


Installing - Usage
----------

Usage

Limitations
